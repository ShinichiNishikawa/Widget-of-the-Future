== Description ==

This plugin adds a widget to display furture scheduled posts list on your sidebar. You can specify how many posts to be listed.

* [GitHub](https://github.com/ShinichiNishikawa/Widget-of-the-Future)
* [Plugin Homepage](http://nskw-style.com/2013/wordpress/plugin-downloads/Widget-of-the-Future.html)

= Title and post counts =

In the widget page, you can specify the title and the number of posts.

== Installation ==

1. Upload `future-posts-widget` folder to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
